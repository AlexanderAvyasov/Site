# GeoEarn Protocol Landing Page

A premium startup landing page for GeoEarn Protocol, a decentralized offline traffic marketplace powered by Solana.

## Overview

GeoEarn connects real-world foot traffic with businesses. Users earn rewards for visiting locations, while businesses pay only for verified visits. Built on Solana for ultra-fast, low-cost transactions.

## Features

- **Premium Web3 Design**: Dark theme with glassmorphism cards, neon gradients, and smooth animations.
- **Animated Elements**: Fade-in sections on scroll, staggered card animations, hover effects, and animated counters.
- **Market Data**: Real numbers on offline advertising market opportunity.
- **Investor-Focused**: Clear business model, token economics, and scalability metrics.
- **Responsive Layout**: Optimized for desktop and mobile devices.
- **Modern Animations**: Intersection Observer for scroll-triggered animations, counter animations.
- **Functional CTAs**: "Join Waitlist" scrolls to form, "Partner With Us" opens email client.

## Structure

1. **Hero**: Animated background, headline, and CTAs.
2. **What This Is**: Core positioning as decentralized marketplace.
3. **How It Works**: End-to-end flow for users, businesses, and on-chain.
4. **How Money Flows**: Step-by-step financial flow.
5. **Market Opportunity**: $600B+ market with real data.
6. **Target Metrics**: Animated counters for users, businesses, visits, volume.
7. **Micro-Traction**: Early pilot results.
8. **Token Model**: Simple explanation of token circulation.
9. **Why Solana**: Product-specific blockchain reasoning.
10. **Trust & Anti-Fraud**: GPS, QR, reputation system.
11. **Use Cases**: Cafes, stores, brands.
12. **How It Looks In Real Life**: Real scenario example.
13. **Roadmap**: Detailed milestones with metrics.
14. **Why Now**: Market timing.
15. **Waitlist**: Email signup form.
16. **Final CTA**: Call to action.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Glassmorphism, gradients, animations, responsive design.
- **JavaScript**: Intersection Observer for scroll animations, counter animations.
- **Font Awesome**: Icons for visual elements.

## Getting Started

1. Open `index.html` in a modern web browser to view the landing page.
2. For development, serve the files using a local server (e.g., `python3 -m http.server 8000`).

## Animations

- **Hero**: Gradient background animation.
- **Sections**: Fade-in on scroll using Intersection Observer.
- **Cards**: Staggered animations and hover scale/glow effects.
- **Counters**: Smooth number animations when in viewport.
- **Buttons**: Scale and glow on hover.

## Customization

- **Content**: Edit `index.html` to modify text and structure.
- **Styling**: Update `styles.css` for colors, fonts, and animations.
- **Animations**: Adjust `script.js` for timing and effects.
- **Metrics**: Change counter targets in HTML data attributes.

## Browser Support

- Modern browsers with CSS Grid, Flexbox, and ES6+ support.
- Fallbacks for older browsers where possible.

## Performance

- Optimized animations using CSS transforms and opacity.
- Lazy loading of animations via Intersection Observer.
- Minimal JavaScript for smooth performance.
